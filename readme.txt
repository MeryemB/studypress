== Description ==

StudyPress is an elearning authoring tool.

== Installation ==

1. Install StudyPress either via the WordPress.org plugin directory, or by uploading the files to your server
2. Activate it
3. That's it. You're ready to go!

== Frequently Asked Questions ==

= How I can include the course that I created in a page or a post ? =

Copy the short code present in the course page and past it in the page or the post that you create.
